Nauka tydzień IV

Git

Repozytorium - projekt (wszystkie jego pliki)
Commit - stan projektu i opis jego zmian

Komendy:
git config --global user.name "nazwa"
git config --global user.email nazwa@email.com
git add . - dodaj wszystkie pliki z obecnego katalogu
get init - dodanie repozytorium gita w katalogu
git status - status zmian
git commit -m "opis zmian" - zarejestrowanie zmian
gitk - otwiera okno historii
git push - wypycha lokalne repozytorium na GitHuba (repozytorium zdalne)
